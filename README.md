# BookStore
基于servlet + jsp + javaBean的网上书城项目
## 主要功能
- 前台
	- 用户登录
	- 用户注册：邮箱激活
	- 书籍检索
	- 购物车
	- 提交订单
	- 支付
- 后台(http://127.0.0.1:8080/BookStore/adminjsps/login.jsp)
	- 书籍管理
	- 分类管理
	- 订单管理
- 后台账号密码
	- 账号：admin
	- 密码：123456
